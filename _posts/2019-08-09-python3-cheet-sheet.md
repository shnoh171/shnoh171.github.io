### Print a + b
```python
if __name__ == '__main__':
    a = int(input())
    b = int(input())
    print(a + b)
```
### If Statement
```python
if __name__ == '__main__':
    left = 10
    right = 20
    if left <= right:
        print("left <= right")
    else:
        print("left > right")
```
